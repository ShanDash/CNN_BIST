# CNN_BIST
Unified CNN + BIST architecture in Verilog HDL targeting Xilinx Artix-7 FPGA. Implements 3×3 convolution, ReLU activation, LFSR test pattern generation and MISR signature analysis for hardware fault detection. Low overhead: 398 LUTs, 36 DSPs, ~58 MHz. | B.Tech Major Project, DTU | Published in IJEER 2026.
